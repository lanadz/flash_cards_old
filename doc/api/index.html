API Documentation

Categories for Student

  • Listing Categories
  • Show Category and returns requested cards from that category

Categories for Teacher

  • Get all categories
  • creates and returns category
  • returns requested cards from selected category

FlashCards for Student

  • Show card

FlashCards for Teacher

  • creates and returns card
  • deletes requested card
  • returns all created cards
  • returns requested card

LearningSessions

  • creates new learning session and returns flash_card_ids
